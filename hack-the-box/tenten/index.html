Hack The Box - Tenten

Quick Summary

Tenten is a medium difficulty machine that requires some outside-the-box/CTF-style thinking to complete. Its IP address is 10.10.10.10. Let’s go to hack!!!

Nmap

As always we will start with nmap to scan for open ports and services:

The nmap report open port 80 with Wordpress 4.7.3 version.

The next step is look if the Wordpress version has known vulnerabilities, then launch wpscan for help to find it.

1.) --url: Target
2.) --api-token: api from https://wpvulndb.com/
3.) --enumerate vp: Search vulnerabilities in plugins.

Was indentified the vulnerability in Job Manager plugin


With the option --enumerate u show the posible users.


When browsing in the web page, found the folders.



With the simple script, can enumerate to find another information, but not show aditional data.
for i in $(seq 1 40); do echo -n "$i: "; curl -s http://10.10.10.10/index.php/jobs/apply/$i/ | grep 'title'; done


Exploit

The version plugin is vulnerable, more information Exploit


Change the extensions files for jpg, jpeg and png.


Run against the target http://10.10.10.10


Found an image vulnerable with possible information, save the image


Running steghide against it with blank password outputs a private key-file, the key is encrypted.

The output is in a file id_rsa, with the sshng2john.py change the format for crack with johnTheRipper.


Copy the sshng2john result to a file id_rsa_exencrypted and run jonh against it.
