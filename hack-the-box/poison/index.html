Hack The Box - Poison

Quick Summary

Poison is a fairly easy machine which focuses mainly on log poisoning and port forwarding/tunneling. The machine is running FreeBSD which presents a few challenges for novice users as many common binaries from other distros are not available. Its IP address is 10.10.10.84 Letâ€™s go to hack!!!

Nmap

As always we will start with nmap to scan for open ports and services:

In this case was used the stylesheet called nmap-bootstrap.xml, which allow to view in a web page the output.

The output show two ports open 80 and 22, but the nmap scan reveal some directories.

File info.php

File phpinfo.php.

File listfile.php in source code.

File browse.php

Enumeration

Intercepting traffic with Burpsuite tool and testing the Local File inclusion vulnerability show the passwd file from server.

Clone repository PayloadAllTheThings and search the file phpinfolfi.py.

Copy the code from php-reverse.php and paste into the phpinfolfi.py. Remember change the IP address with own.

Change the file with vulnerability local file inclusion, in this case browse.php

Exploit

Run the netcat listener and launch the phpinfolfi.py.

The output reveal an error on tcp_name section, locate and change with gt letters in the whole code.

Run again the code.

We have a reverse connection over netcat listener.

The session no has root privileges.

Enumerate the system into the each one folders.

The file pwdbackup.txt show a code encrypted 13 times with base64.

Copy the code and decrypted.

Connect by ssh with the password found.

In the current directory there is a file secret.zip, download from server.

Enumerate the system, show software as VNC.

The Vnc is a local service, the sockstat -4l show the 127.0.0.1:5901 and 127.0.0.1:5801.

Port-Forwarding

The SSH protocol is the best way for Port-Forwarding, in the current session SSH press ~C for entry in the ssh mode and then type -D 1080.

In the Kali linux, run netstat -punta command for review connections and reveal the port 1080 for port forwarding.

Capture The Flag

Run the next commando for vnc viewer.

A pop-up show the vnc with root privileges.

And we owned root !
